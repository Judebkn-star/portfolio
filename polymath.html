Ce que fait Polymath

01
Leaderboard des "Sharps" en temps réel

Agrège les données des 200+ meilleurs traders Polymarket via l'API officielle. Filtrage par 9 catégories (Politique, Crypto, Sport, Tech...) × 4 périodes (24h, 7j, 30j, All Time). Chaque trader affiche son ROI, profit total, volume et win rate.

Polymarket APIPagination9 catégoriesCache 5min
02
Recommandations personnalisées (Advisor)

Un algorithme de scoring sur 100 points analyse le profil utilisateur — capital disponible, appétit au risque (conservateur/équilibré/agressif), domaines d'intérêt — pour recommander les traders les plus adaptés et calculer le stake optimal à miser sur chaque pari.

Scoring 4 facteursCapital MatchRisk ProfileDomain Match
03
Détection automatique de bots

Système heuristique qui identifie les traders automatisés : win rate anormal (95%+), timing trop régulier, taille des trades uniforme, fréquence excessive. Chaque wallet reçoit un score de suspicion 0-100% avec un flag "Bot probable" si score > 75%.

HeuristiquesScore 0-100%Cache 24h
04
Profil détaillé & Copy Trading

Page dédiée à chaque trader : graphiques de performance, historique des trades, positions ouvertes, domaines de spécialisation. Intégration d'un bouton "Copy Trade via TradeFox" pour répliquer directement les positions des meilleurs sharps.

Algorithme de scoring des recommandations (Advisor) TypeScript 
function scoreSharp(sharp, profile) {
  let score = 0;

  // Capital match (35 pts)
  const tier = getCapitalTier(profile.capital);
  if (sharp.totalVolume >= tier.min && sharp.totalVolume <= tier.max)
    score += 35;

  // Risk profile (30 pts)
  if (profile.risk === "aggressive") score += sharp.roi * 0.3;
  if (profile.risk === "conservative") score += sharp.winRate * 0.3;

  // Domain matching (25 pts)
  const overlap = profile.domains.filter(d => sharp.domains.includes(d));
  score += (overlap.length / profile.domains.length) * 25;

  // Bonus profitabilité (10 pts)
  if (sharp.totalProfit > 10000) score += 10;

  return Math.min(score, 100);
}
Détection de bot par analyse comportementale TypeScript 
function detectBot(trades) {
  let suspicion = 0;

  const winRate = trades.filter(t => t.profit > 0).length / trades.length;
  if (winRate > 0.95) suspicion += 40;

  const intervals = trades.map((t, i) =>
    i > 0 ? t.timestamp - trades[i-1].timestamp : null
  ).filter(Boolean);
  const stdDev = getStdDev(intervals);
  if (stdDev < 60) suspicion += 30; // trop régulier

  const sizes = trades.map(t => t.amount);
  if (getStdDev(sizes) < 5) suspicion += 20; // trop uniforme

  return { score: suspicion, isBotProbable: suspicion > 75 };
}
