Pourquoi ce projet ?

Ce projet est avant tout un terrain de jeu personnel — la F1 est une passion, et quoi de mieux qu'un sujet qu'on aime pour s'entraîner sérieusement ? L'idée : utiliser un dataset riche et complexe pour pratiquer des compétences directement transférables en entreprise. Modélisation relationnelle, requêtes SQL avancées, storytelling data, visualisation — tout y est.

Démarche stratégique

Chaque analyse répond à une vraie question business : quels facteurs expliquent la domination d'une écurie ? Comment mesurer l'impact d'une stratégie pit stop sur le résultat final ?

Base de données SQLite multi-tables : courses, pilotes, constructeurs, résultats, pit stops, qualifications, classements — plus de 500 000 lignes croisées.

SQL d'abord pour explorer et modéliser, Python pour aller plus loin, puis mise en récit des données pour rendre les insights accessibles à un public non-technique.

CTEs, window functions, jointures multi-tables, agrégations conditionnelles, détection de tendances, corrélation — dans un contexte concret et engageant.

Quelques requêtes SQL

Top 10 pilotes — victoires cumulées sur toute la carrière SQL 
-- Classement des pilotes par nombre de victoires totales
SELECT
    d.forename || ' ' || d.surname  AS pilote,
    COUNT(r.resultId)               AS victoires,
    RANK() OVER (
        ORDER BY COUNT(r.resultId) DESC
    )                               AS classement
FROM results r
JOIN drivers d ON r.driverId = d.driverId
WHERE r.position = 1
GROUP BY d.driverId
ORDER BY victoires DESC
LIMIT 10;
Temps moyen des pit stops par écurie et par saison SQL 
-- CTE : durée moyenne des pit stops par constructeur
WITH pit_perf AS (
    SELECT
        con.name                          AS ecurie,
        ra.year                           AS saison,
        ROUND(AVG(ps.milliseconds) / 1000.0, 2) AS duree_moy_sec,
        COUNT(*)                          AS nb_arrets
    FROM pit_stops ps
    JOIN results      re  ON ps.raceId = re.raceId
                         AND ps.driverId = re.driverId
    JOIN constructors con ON re.constructorId = con.constructorId
    JOIN races        ra  ON ps.raceId = ra.raceId
    GROUP BY con.name, ra.year
)
SELECT *
FROM pit_perf
WHERE nb_arrets > 10
ORDER BY saison DESC, duree_moy_sec ASC;
